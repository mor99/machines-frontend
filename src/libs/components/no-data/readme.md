# 使用范围
  列表无数据

# API

| 属性        | 说明    |  类型  |  默认值  |
| --------   | -----:  | :----: | :----:  |
| text     | 显示文字 |  string   |暂无数据|



# usage

```javascript
  import NoData from "path-to-component";

  class Index  extends PureComponent {
    render() {
      return (
        <div>
          <NoData />
        </div>
      )
    }
  }
```
